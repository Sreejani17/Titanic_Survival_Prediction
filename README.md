# 🚢 Titanic Survival Prediction

A beginner supervised machine learning project that predicts whether a 
passenger survived the Titanic disaster, using Logistic Regression.

## 📊 Dataset
Titanic dataset from Kaggle ("Titanic - Machine Learning from Disaster").
Contains passenger details like age, sex, ticket class, and fare.

## 🔧 Tech Stack
- Python
- pandas
- scikit-learn

## 🧠 Approach
1. Loaded and explored Titanic passenger data
2. Cleaned missing values (Age, Embarked) and dropped unhelpful columns
3. Converted categorical features (Sex, Embarked) into numbers
4. Split data into training and test sets
5. Trained a Logistic Regression model
6. Evaluated using Accuracy, Confusion Matrix, and Classification Report
